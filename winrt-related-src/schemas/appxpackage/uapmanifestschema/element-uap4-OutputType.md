
| Attribute | Description | Data type | Required | Default value |
|-|-|-|-|-|
| **SubType** | Either a GUID or a format type of the codec. | Either a 4 character code or a GUID in the form xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx. | Yes |  |
