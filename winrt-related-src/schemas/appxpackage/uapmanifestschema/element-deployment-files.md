# deployment:Files

Contains one or more **File** elements specifying DLL files that provide DEH functionality that ships outside of an OS release.

## Syntax

```xml
<deployment:Files>

  <!-- Child elements -->
  deployment:Files {0,1000}
</deployment:Files>
```

### Key

`{}`   specific range of occurrences
