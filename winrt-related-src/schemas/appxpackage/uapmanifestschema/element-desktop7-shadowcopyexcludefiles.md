# desktop7:ShadowCopyExcludeFiles

Specifies a set of files to be excluded by the Volume Shadow Copy Service (VSS).

## Syntax

```xml
<desktop7:ShadowCopyExcludeFiles>

    desktop7:ShadowCopyExcludeFile{0,1000}

</desktop7:ShadowCopyExcludeFiles>
```

### Key

`{}` specific range of occurrences
